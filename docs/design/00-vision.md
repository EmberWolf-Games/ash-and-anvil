# Ash & Anvil (1st Edition) — Vision

## Pillars

1. **Forge your hero** — Character creation is intentional, visual, and completed inside Foundry.
2. **Clear d20 play** — Roll a d20, add relevant bonuses, compare to a Difficulty; Boon and Bane modify dice, not math soup.
3. **DM authority** — World settings and rules profiles tune automation and numeric knobs without hacking the system.
4. **Original fantasy** — Setting, names, and mechanics are EmberWolf IP; not a clone of any published RPG.

## 1st Edition scope

| In scope (MVP) | Later milestones |
|----------------|------------------|
| Level-1 character creation | Combat loop |
| Ancestry, class, background | Arcane magic |
| Skills & six attributes | Psionics |
| Derived HP, proficiency, initiative | Divine gifts |
| Foundry sheets + chargen wizard | Monsters & GM tools |

## Table expectations

- **Grid:** 5 ft squares (configurable in Foundry scenes).
- **Party size:** 3–6 players.
- **Tone:** Heroic fantasy with room for gritty play via rules profile.

## Publication

- System package distributed via GitHub releases and Forge Bazaar manifest.
- Rules text lives in `docs/design/` during development; player-facing books are a separate future product.
